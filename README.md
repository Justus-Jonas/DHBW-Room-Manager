# dhbw room manager

**Prerequisites**

 * Python 3.7.3
 * Django 2.2.1
 * pip 19.1.1 

**How to run this project?**

1.  `pip3 install apscheduler icalendar`
2.  `python3 manage.py makemigrations`
3.  `python3 manage.py migrate`
4.  `python3 manage.py runserver`

(First start will take a while to download stuff.)
